Enum sage::asm::location::Location

source · 
pub enum Location {
    Address(usize),
    Indirect(Box<Self>),
    Offset(Box<Self>, isize),
}
Expand description

A location in memory (on the tape of the virtual machine).

Variants§

§

Address(usize)

A fixed position in the tape (a constant address known at compile time).

§

Indirect(Box<Self>)

Use the value of a cell on the tape as an address. For example, Indirect(SP) is the location of the top item on the stack.

§

Offset(Box<Self>, isize)

Go to a position in memory, and then move the pointer according to an offset. For example, Offset(Address(8), -2) is equivalent to Address(6).
