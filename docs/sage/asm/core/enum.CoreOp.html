Expand description

A core instruction of the assembly language. These are instructions guaranteed to be implemented for every target possible.

Variants§

§

Comment(String)

§

Many(Vec<CoreOp>)

Many instructions to execute; conveniently grouped together. This is useful for code generation.

§

Global

Declare a global variable.

This will allow the program to reference the global variable using $name in the assembly code, where name is the name of the global variable.

To access the first element of the global variable, use $name. To access the second element of the global variable, use $name + 1.

To get the address of the first element of the global variable, use lea $name, $tmp.
